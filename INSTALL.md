# Installation Guide

## Quick Install

```bash
pip install mlflow-descope-auth
```

## From Source

```bash
git clone https://github.com/descope/mlflow-descope-auth.git
cd mlflow-descope-auth
pip install -e .
```

## With Development Dependencies

```bash
pip install -e ".[dev]"
```

## Verify Installation

```bash
python -c "import mlflow_descope_auth; print(mlflow_descope_auth.__version__)"
```

Expected output: `0.1.0`
